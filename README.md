# Climate-Smart Agriculture Planner

A Streamlit application for analyzing climate change impacts on African agriculture using Shared Socioeconomic Pathways (SSPs).

## Features

- **SSP Scenario Analysis**: Compare different socioeconomic pathways
- **Climate Projections**: Temperature and precipitation changes through 2100
- **Yield Impact Assessment**: Crop yield projections under climate change
- **Regional Vulnerability**: Identify high-risk areas
- **Policy Recommendations**: Adaptation strategies for each scenario
